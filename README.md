# Landing Page - The End of Alzheimer's

## Como iniciar

### Opção 1: Abrir direto no navegador
Dê dois cliques no arquivo `index.html`

### Opção 2: Usar servidor local
```bash
npx serve
```
Acesse: http://localhost:3000

---

**Arquivos:**
- `index.html` - Página principal
- `video.png` - Thumbnail do vídeo
- `dtc.png` - Imagem da oferta (aparece após 15s)
